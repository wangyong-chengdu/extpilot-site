Slides Timer for Google Slides — Last updated: April 23, 2026

Overview

Slides Timer for Google Slides helps users display countdown timers, clocks, and dates inside Google Slides presentations. Core functionality runs locally in the browser and does not require an account.

Data We Access

  • Google Slides page content needed to detect timer tags and render live output during presentation mode
  • Local browser storage for install date, usage count, and rating prompt state

Data We Do Not Collect

  • No browsing history
  • No Google account data
  • No analytics or telemetry backend
  • No sale or sharing of personal data

Permissions

The extension currently uses only the storage permission to save local extension state.
