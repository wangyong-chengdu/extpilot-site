Direct answer

If you want to add a countdown timer to Google Slides, type a tag like <<5:00->> into any text box and start presentation mode. Slides Timer turns that tag into a live countdown, clock, or date while you present.

How it works

1. Open Slides

Open any Google Slides presentation.

2. Type a tag

Try <<5:00->>, <<time>>, or <<date>>.

3. Start presentation mode

The tag is replaced with live output while you present.

4. Control it live

Press Y to pause or resume, R to reset.

Common tags

<<5:00->>

5-minute countdown timer.

<<2:00+>>

Count-up stopwatch.

<<time>>

Current time during presentation.

<<date>>

Current date during presentation.

FAQ

Why do I only see tags in edit mode?

That is expected. Live timers appear only in presentation mode.

Do I need an account or paid plan?

No. Core usage does not require an account, and the product strategy for this extension avoids a premium paywall.
