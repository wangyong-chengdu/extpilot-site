Frequently Asked Questions

Does this work with private accounts?

You can only export followers/following from accounts you can already see on Instagram. For private accounts, you need to be an approved follower.

Will my Instagram account get banned?

IG Follower Exporter uses Instagram's own API with built-in rate limiting to stay within safe usage patterns. However, excessive use of any automation tool carries some risk.

What data is collected?

None. All processing happens locally in your browser. We do not collect, store, or transmit any Instagram data. The only network request is license key verification for Pro users.

Can I export more than 10,000 followers?

Yes. Pro mode supports large exports with automatic checkpoint and resume. The export may take longer for very large accounts due to Instagram's rate limits.
